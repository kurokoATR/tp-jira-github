# tp-jira-github
Trabajo práctico UTN GitHub y Jira

Integrante
solo Tomás Aguirre 

Escenario 
Análisis de ventas de una pequeña empresa

Sobre el proyecto

Este trabajo fue realizado para poner en práctica herramientas de trabajo colaborativo como Git GitHub y Jira
La idea principal del proyecto es analizar un conjunto de datos de ventas usando Python, generando algunos resultados básicos y simples para entender mejor la información
También se trabajó con control de versiones y organización de tareas para simular un entorno real

Estructura del repositorio

/datos  archivos de datos usados en el análisis
/scripts código Python del proyecto
/resultados  gráficos y resultados generados

Herramientas usadas

Python
Git
GitHub
Jira
Google Colab

Funcionalidades

calcular ventas totales
obtener el producto más vendido
analizar ventas por mes
generar gráficos básicos

Cómo ejecutar el proyecto

Ejecutar el archivo ubicado en la carpeta /scripts.

El proyecto fue pensado para funcionar correctamente desde Google Colab.
Proyecto realizado utilizando GitHub, Jira y Google Colab.
